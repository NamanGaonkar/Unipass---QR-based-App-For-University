# UniPass Student App

This app is only for students.

- No password login
- Scan teacher QR (or paste QR JSON)
- Sends attendance proof to local campus backend

## Run

1. `cd student-mobile`
2. `npm install`
3. `npm run android`

Backend must run on `http://localhost:5000`.
For Android emulator this app already uses `http://10.0.2.2:5000`.
